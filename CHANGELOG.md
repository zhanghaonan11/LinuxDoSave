# Changelog

本文件按 GitHub Releases 风格整理 `LinuxDoSave` 的版本演进。

## [f] - 2026-01-06

**标签**：`f`  
**提交**：`f0b3137`

### Added

- 增加对站内单页应用路由切换的监听，进入新帖子页后可再次自动注入按钮

### Changed

- 脚本匹配范围从 `https://linux.do/t/*` 扩展到 `https://linux.do/*`
- `Ctrl+S` / `Cmd+S` 快捷键仅在帖子页生效，避免影响站内其他页面

## [e] - 2026-01-04

**标签**：`e`  
**提交**：`d43c9cc`

### Added

- 复制和下载内容中加入当前帖子 URL
- 增加 `upload://...` 图片链接解析与替换逻辑

### Changed

- 复制结果从“标题 + Markdown 源码”升级为“标题 + URL + Markdown 源码”
- 下载按钮提示文案调整为更直接的 `.md` 下载说明

## [d] - 2026-01-04

**标签**：`d`  
**提交**：`5a792b6`

### Added

- 增加独立下载按钮
- 增加 `Ctrl+S` / `Cmd+S` 快捷键下载能力
- 增加文件名清理逻辑，避免下载文件名包含非法字符

### Changed

- 将“获取数据”和“执行动作”拆分，便于复制与下载共用同一份帖子数据

## [c] - 2026-01-04

**标签**：`c`

- 复制时自动在 Markdown 源码前拼接一级标题

### Changed

- 按钮语义从“复制主贴源码”升级为“复制标题和源码”

## [b] - 2026-01-04

**标签**：`b`  
**提交**：`6946d68`

- 增加请求 `/raw/{topicId}/1` 的原始 Markdown 获取逻辑

- UI 从显式下载按钮改为更轻量的图标型复制按钮
